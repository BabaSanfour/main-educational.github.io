# Venue

The workshop will be located at in pavillion Roger Gaudry at [Université de Montréal](https://www.umontreal.ca/). Registration and food will be set up in the main entrance hall. The single track sessions will run in P-310, while parallel sessions will run both in P-310 and S-142.
